AI API 키 설정

BookEngine은 BYOK(Bring Your Own Key) 방식으로 동작합니다. Google Gemini 또는 OpenAI의 API 키를 직접 발급받아 입력해야 합니다. 키는 로컬 PC에만 저장되며 외부 서버로 전송되지 않습니다.

BYOK 정책

키 보안 정책
입력한 API 키는 BookEngine 앱 내부의 암호화된 로컬 저장소에만 저장됩니다. BookEngine 서버, 개발사 또는 제3자에게 절대 전송되지 않으며, 모든 AI 요청은 사용자의 PC에서 직접 Google/OpenAI 서버로 전송됩니다.

이 방식의 장점은 사용량에 따른 정확한 비용 통제가 가능하고, 데이터 프라이버시가 완전히 보장된다는 점입니다. BookEngine 자체는 AI 처리에 대해 별도 마진을 부과하지 않습니다.

지원 AI 서비스

G
Google Gemini

Gemini 2.0 Flash 및 Gemini 2.0 Pro를 지원합니다. 빠른 처리 속도와 긴 컨텍스트 창(최대 1M 토큰)으로 장편 원고 집필에 적합합니다.

Google AI Studio 열기
O
OpenAI

GPT-4o 및 GPT-4o mini를 지원합니다. 고품질 문장 생성과 정밀한 편집 작업에 적합합니다. Gemini와 혼용 사용도 가능합니다.

OpenAI Platform 열기

Google Gemini API 키 발급

1

Google AI Studio 방문

브라우저에서 aistudio.google.com에 접속합니다. Google 계정으로 로그인합니다. (Google 계정이 없다면 먼저 생성하세요.)

2

API 키 생성

좌측 메뉴에서 "Get API key"를 클릭합니다. "Create API key" 버튼을 누르면 새 키가 즉시 생성됩니다. 키는 AIza...로 시작하는 39자 문자열입니다.

3

키 복사 및 저장

생성된 키를 복사합니다. 이 키는 한 번만 표시되므로 안전한 곳에 보관하세요. 분실 시에는 새 키를 생성하고 BookEngine에 다시 입력하면 됩니다.

무료 사용량
Gemini API는 Gemini 2.0 Flash 기준 월 50만 토큰을 무료로 제공합니다 (2026년 기준). 단편 도서(5만 단어 이하)는 무료 한도 내에서 처리 가능할 수 있습니다. Google AI Studio에서 현재 사용량과 한도를 확인하세요.

OpenAI API 키 발급

1

OpenAI Platform 방문

platform.openai.com에 접속 후 로그인합니다. 계정이 없다면 회원가입을 먼저 진행하세요. 결제 수단 등록이 필요합니다.

2

API Keys 메뉴 이동

상단 메뉴에서 "API keys"를 클릭합니다. "+ Create new secret key" 버튼을 눌러 새 키를 생성합니다. 키 이름을 BookEngine으로 지정하면 나중에 관리가 편리합니다.

3

키 복사

생성된 키(sk-...로 시작)를 복사합니다. 이 화면을 닫으면 다시 확인할 수 없으므로 즉시 복사해두세요.

BookEngine에서 API 키 입력

1

설정 화면 열기

BookEngine 앱 우측 상단의 설정 아이콘(톱니바퀴)을 클릭합니다. 또는 메뉴바에서 "설정 → API 키 관리"로 이동합니다.

2

키 입력 및 검증

"Gemini API 키" 또는 "OpenAI API 키" 입력란에 복사한 키를 붙여넣습니다. "검증" 버튼을 클릭하면 실제 API 호출로 키의 유효성을 즉시 확인합니다. 검증 성공 시 녹색 체크 표시가 나타납니다.

3

기본 모델 선택

스테이지별 기본 AI 모델을 선택합니다. 초안/기획 스테이지는 Gemini 2.0 Flash, 편집/교정 스테이지는 GPT-4o를 권장합니다. 또는 모든 스테이지에 동일한 모델을 적용할 수 있습니다.
