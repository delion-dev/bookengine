자주 묻는 질문

월 구독이 없나요?

네. BookEngine은 월정액 구독 없이 1회 구매 후 영구 사용합니다. 업데이트는 구매 후 1년(개인) / 2년(팀) 포함이며, 이후에는 선택적으로 업그레이드할 수 있습니다.

AI API 비용은 별도인가요?

BookEngine은 BYOK(Bring Your Own Key) 방식으로 동작합니다. Google Gemini / OpenAI API 키를 직접 발급해 입력하면 해당 계정으로 요금이 청구됩니다. 일반적인 도서 1권 집필 시 Gemini Flash 기준 약 $1~3 수준입니다.

인터넷 연결이 필요한가요?

라이선스 활성화는 최초 1회 온라인 인증이 필요합니다. 이후 집필 작업은 완전 오프라인 모드로 동작하며, AI API 호출 시에만 인터넷을 사용합니다.

Mac / Linux는 지원하나요?

현재 Windows 10/11만 지원합니다. Mac / Linux 버전은 v0.2.0 로드맵에 포함되어 있으며 출시 알림은 GitHub Releases를 통해 받을 수 있습니다.

환불 정책이 어떻게 되나요?

구매 후 14일 이내, 도서 생성 횟수 3회 미만인 경우 전액 환불을 지원합니다. 문의는 hello@bookengine.io로 보내주세요.

라이선스 키는 어디서 받나요?

구매 완료 후 이메일로 라이선스 키(BKENG-XXXXX-XXXXX-XXXXX-XXXXX 형식)가 발송됩니다. BookEngine 앱 최초 실행 시 키 입력 화면에 붙여넣기 하면 즉시 활성화됩니다.
