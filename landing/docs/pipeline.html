AI 집필 파이프라인

BookEngine의 13단계 자동화 파이프라인은 기획안 입력부터 EPUB 패키징까지 전 과정을 처리합니다. 각 스테이지는 독립적으로 실행하거나, 전체를 순차적으로 자동 진행할 수 있습니다.

파이프라인 개요

각 스테이지는 이전 스테이지의 출력을 입력으로 받아 처리합니다. 게이트 검증에 실패한 스테이지는 자동으로 재시도(Self-Healing)되며, 결과는 실시간 대시보드에서 확인할 수 있습니다.

Work Order 시스템
BookEngine은 챕터 단위 병렬 처리를 지원합니다. 예를 들어 S4(초안 집필) 단계에서 20개 챕터를 동시에 AI에 요청하여 처리 시간을 대폭 단축합니다.

준비 단계

S-1

사전구성 Pre-Config

책의 기본 메타데이터를 설정합니다. 제목, 부제, 저자, 언어, 장르, 목표 챕터 수, 챕터당 목표 단어 수를 입력합니다. AI 모델 및 온도(Temperature) 값도 이 단계에서 설정합니다.

S0

초기화 Initialize

작업 폴더 구조를 생성하고 기획안 파일을 파싱합니다. 챕터 목록 초안, 전체 컨텍스트 문서, 스타일 프로파일을 생성합니다. 이후 모든 스테이지가 참조하는 book_context.json이 이 단계에서 만들어집니다.

기획 단계

S1

기획 Planning

기획안을 분석해 책의 핵심 주제, 독자층, 차별점을 정의합니다. AI가 기획의 강점과 보완점을 평가하고 최종 집필 방향성 문서를 생성합니다.

S2

목차 TOC

챕터 구성과 절(Section) 구조를 생성합니다. 각 챕터의 핵심 내용, 학습 목표, 예상 단어 수를 포함한 상세 목차가 출력됩니다. 사용자가 목차를 검토하고 수정한 뒤 다음 단계로 진행합니다.

S3

개요 Outline

목차의 각 챕터를 상세 개요로 확장합니다. 절별 핵심 포인트, 예시 코드/이미지 앵커, 인용 슬롯을 포함한 챕터 집필 지시서(Chapter Brief)가 생성됩니다.

집필 단계

S4

초안 집필 Draft

챕터 개요를 기반으로 AI가 전체 초안을 집필합니다. 챕터는 병렬로 처리되며, 단어 수 게이트(목표의 80% 이상)를 통과한 챕터만 다음 단계로 진행합니다. 게이트 실패 시 자동으로 보강 집필(Re-draft)이 실행됩니다.

S5

자료 수집 Research

초안에 삽입된 앵커 슬롯([REF:xxx])을 분석해 참고 자료를 수집합니다. 웹 검색 기반 인용 자료, 통계, 사례 연구를 자동으로 수집하여 초안에 삽입합니다.

S6

이미지 자산 Images

초안에 삽입된 이미지 앵커([IMG:xxx])에 맞는 이미지를 수집하거나 생성합니다. 다이어그램, 플로우차트는 Mermaid 코드로 자동 생성되며, PNG로 렌더링됩니다.

S6B

이미지 보정 Image QA

수집된 이미지의 저작권, 해상도, 파일 크기를 검증합니다. 기준 미달 이미지는 대체 이미지 탐색 또는 AI 생성 이미지로 교체됩니다.

편집 및 교정 단계

S7

편집 Edit

챕터 전체를 대상으로 구조 편집을 수행합니다. 흐름 개선, 중복 제거, 예시 강화, 단락 재구성을 AI가 처리합니다. 편집 전후 단어 수 비교 리포트가 자동으로 생성됩니다.

S8

교정 Proofread

맞춤법, 문법, 일관성(용어, 표기법)을 검사합니다. 7가지 QA 게이트(단어 수, 앵커 슬롯, 이미지 태그, 번호 매기기, 참고 문헌, 제목 계층, 코드 블록)를 자동으로 검증합니다.

S8A

심화 교정 Deep QA

S8 교정 이후에도 남아있는 구조적 문제를 심층 분석합니다. 챕터 간 흐름 일관성, 용어 사전 위반, 독자 수준 적합성을 종합 평가하고 최종 교정 리포트를 생성합니다.

S9

출판 원고 Final

모든 챕터를 하나의 마스터 마크다운 파일로 통합합니다. 목차, 서문, 저자 소개, 참고 문헌 페이지를 자동으로 추가합니다. 최종 단어 수, 챕터 수, QA 통과율이 기록된 원고 완성 리포트가 생성됩니다.

Google Books 출판 단계

S9 완성 원고를 Google Books에 출판하기 위한 추가 스테이지입니다.

S10

스타일 가이드 Style

출판 대상에 맞는 스타일 가이드를 선택합니다. GBOOK-TECH(기술서), ACAD(학술서), BUSI(비즈니스), NFIC(논픽션), TUTO(튜토리얼), MINI(미니북), CUSTOM(사용자 정의) 중 선택합니다. 선택된 스타일 가이드가 전체 원고에 적용되어 폰트, 여백, 제목 스타일이 통일됩니다.

S11

EPUB 패키징 EPUB

완성된 원고를 EPUB 3.0 표준으로 패키징합니다. ISBN-13, Google Books ID, 메타데이터, SEO 키워드(AI 자동 생성 7개 + 롱테일 5개), 커버 이미지를 포함한 완성형 .epub 파일이 생성됩니다. 생성된 파일은 Google Books Partner Center에 바로 업로드할 수 있는 규격입니다.
