시작하기

BookEngine 설치 및 초기 설정 가이드입니다. 아래 단계를 순서대로 따라하면 15분 안에 첫 번째 AI 집필을 시작할 수 있습니다.

시스템 요구사항

BookEngine은 로컬 설치형 데스크톱 앱입니다. 아래 환경이 갖춰져 있어야 정상 동작합니다.

항목 최소 요구사항 권장 사양
운영체제 Windows 10 (64비트) Windows 11
RAM 8 GB 16 GB 이상
저장공간 4 GB 여유 공간 SSD 10 GB 이상
인터넷 AI API 호출 시 필요 상시 연결 권장
기타 — WebView2 런타임 (자동 설치)
AI API 키 필요
BookEngine은 BYOK(Bring Your Own Key) 방식입니다. Google Gemini 또는 OpenAI API 키가 반드시 필요합니다. 키 발급 방법은 API 키 설정 가이드를 참고하세요.

설치 단계

BookEngine 설치는 다음 4단계로 완료됩니다.

1

설치 파일 다운로드

다운로드 페이지에서 BookEngine-Setup-x.x.x.msi 파일을 내려받습니다. 파일 크기는 약 80~120 MB이며, Windows Installer 패키지 형식입니다.

2

설치 실행

다운로드한 .msi 파일을 실행합니다. "Windows의 PC 보호" 경고가 표시될 경우 "추가 정보" → "실행"을 클릭하세요. 설치 마법사의 안내에 따라 설치 경로를 선택하고 완료합니다.

3

라이선스 키 입력

앱을 처음 실행하면 라이선스 활성화 화면이 나타납니다. 구매한 라이선스 키(XXXX-XXXX-XXXX-XXXX 형식)를 입력하고 "활성화" 버튼을 누릅니다. 무료 체험판은 키 없이 30일간 사용할 수 있습니다.

4

AI API 키 설정

라이선스 활성화 후 설정 화면으로 이동합니다. "API 키 관리" 탭에서 Google Gemini 키 또는 OpenAI 키를 입력하세요. 키는 로컬 PC에만 저장되며 외부로 전송되지 않습니다. 자세한 방법은 API 키 설정 가이드를 참고하세요.

첫 번째 책 등록

설치가 완료되면 대시보드에서 새 책 프로젝트를 생성할 수 있습니다.

1

책 이름 입력

대시보드 우측 상단의 "새 책 만들기" 버튼을 클릭합니다. 책 제목과 부제를 입력합니다. 나중에 수정할 수 있으므로 임시 제목도 괜찮습니다.

2

작업 폴더 선택

책 원고 파일이 저장될 폴더를 선택합니다. 폴더 탐색기가 열리면 원하는 위치를 지정하세요. BookEngine이 해당 폴더 내에 챕터별 마크다운 파일을 자동으로 생성합니다.

3

기획안 파일 업로드

AI 파이프라인이 책의 방향성을 이해할 수 있도록 기획안 파일을 업로드합니다. 지원 형식: .txt, .md, .docx. 기획안이 없다면 BookEngine의 인터랙티브 기획 마법사를 이용해 생성할 수 있습니다.

설정 완료
책 등록이 완료되면 AI 집필 파이프라인 대시보드로 이동합니다. S-1 단계(사전구성)부터 순서대로 실행하거나, 원하는 스테이지를 선택해 실행할 수 있습니다. 전체 파이프라인 흐름은 AI 집필 파이프라인 가이드를 참고하세요.
