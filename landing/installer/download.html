설치 방법

  1. 1
    설치 파일 다운로드 위 버튼을 클릭하여 설치 파일을 다운로드합니다.
  2. 2
    설치 마법사 실행 다운로드된 .exe 또는 .msi 파일을 실행합니다. Windows SmartScreen 경고가 뜨면 "추가 정보 → 실행"을 선택하세요.
  3. 3
    AI API 키 준비 Google Gemini API 키 또는 OpenAI API 키가 필요합니다. 앱 실행 후 설정에서 입력하세요.
  4. 4
    라이선스 키 입력 (선택) 무료 체험 모드로 바로 시작하거나 라이선스 키를 입력하여 전체 기능을 활성화할 수 있습니다.
  5. 5
    첫 번째 책 등록 기획안 파일과 작업 폴더를 지정하면 자동 집필 파이프라인이 시작됩니다.
